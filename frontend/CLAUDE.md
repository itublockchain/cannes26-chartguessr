# Project Rules

## Tech Stack
- React 19 + TypeScript + Vite
- This is a **frontend-only** project. Do NOT write any backend code (no servers, no APIs, no databases).

## Language
- All UI text, error messages, placeholders, button labels, and content MUST be in **English**.

## Responsive & Screen Size
- **Minimum supported screen width: 800px**
- Do NOT write responsive CSS for viewports below 800px.
- For screens under 800px, display a **full-screen overlay warning** that blocks all content:
  - Message: "This application is only supported on screens 800px and above."
  - The overlay must completely cover the page so the user cannot interact with anything underneath.
